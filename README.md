## Key Features

- Developed according to Figma designs, the landing page brings together resources and design experts
- User-focused, intuitive design
- Optimized the landing page for various screen sizes
- JavaScript integration is planned for future updates to improve interactivity and user experience

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/MarinaDiana01/landing-page.git
   cd landing-page
   ```

2. **Run the project:**

   Open `index.html` in your browser, or use a **Live Server** extension for better development experience.
